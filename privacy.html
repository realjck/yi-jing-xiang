Politique de Confidentialité

Dernière mise à jour : Avril 2026

1. Préambule

Cette application de tirage du Yi Jing est conçue comme un outil de consultation documentaire et personnel. Le respect de votre vie privée est au cœur de notre démarche : l'application fonctionne de manière autonome et ne collecte aucune donnée personnelle.

2. Collecte et Traitement des Données

  • Données Personnelles : Aucune donnée nominative (nom, email, adresse) n'est demandée, collectée ni stockée.
  • Historique des Tirages : Vos tirages et interprétations sont traités exclusivement en local sur votre appareil. Ils ne sont jamais transmis à un serveur tiers.
  • Cookies et Traceurs : L'application n'utilise aucun cookie publicitaire ni aucun outil d'analyse d'audience (type Google Analytics).

3. Fonctionnement de la PWA (Stockage Local)

Pour permettre un fonctionnement hors-ligne et mémoriser vos préférences, l'application utilise des technologies de stockage local (comme le LocalStorage ou l'IndexedDB) sur votre navigateur.

  • Ces données restent physiquement sur votre appareil.
  • Elles ne sont accessibles que par vous à travers ce navigateur.
  • Vous pouvez les supprimer à tout moment en vidant le cache de votre navigateur.

4. Hébergement

L'application est un site statique hébergé sur GitHub Pages. GitHub peut enregistrer des logs techniques anonymes (adresse IP, type de navigateur) pour des raisons de sécurité et de maintenance, conformément à sa propre politique de confidentialité, indépendamment de la logique de cette application.

5. Contact

Pour toute question concernant cette application ou son code source, vous pouvez contacter le développeur via le dépôt officiel du projet sur GitHub.

Privacy Policy

Last updated: April 2026

1. Overview

This YiJing application is a static documentary tool. We are committed to digital sobriety and user privacy. This app is designed to function autonomously and does not collect any personal data.

2. Data Collection

  • Personal Information: No personal data (name, email, address) is requested, collected, or stored.
  • Usage Data: We do not use any analytics or tracking tools (such as Google Analytics).
  • Readings History: Your hexagram readings are processed locally on your device. They are never transmitted to any remote server.

3. Local Storage (PWA)

To enable offline functionality and save your preferences, the application uses local browser storage technologies (such as LocalStorage or IndexedDB).

  • This data remains on your device.
  • It is only accessible by you through your browser.
  • You can clear this data at any time by emptying your browser cache or resetting the application.

4. Hosting

The application is a static site hosted on GitHub Pages. GitHub may collect basic technical logs (IP address, browser type) for security and maintenance purposes, in accordance with their own privacy policy and independent of this application's logic.

5. Contact

If you have any questions regarding this application, you may contact the developer via the official project repository on GitHub.
